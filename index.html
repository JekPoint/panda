熊猫书签 / Panda Bookmark
广受绅士喜爱的网页脚本
A script designed for hentai

步骤 1 / Step 1
[exkey]
请绑定exkey，生成私人书签。（留空使用共享账号）
Bind exkey to generate private bookmark. (Leave blank to use shared account)

步骤 2 / Step 2

将上面按钮“[panda]”拖入书签栏。
Pull the button "[panda]" into Bookmarks Toolbar.

Android请使用Firefox，长按按钮“[panda]”，为此链接添加书签。
For Android, please install Firefox, long press button "[panda]", Bookmark Link.
IOS请全选复制代码，然后将本页面添加书签。点击地址栏，长按书签图标编辑，清空地址并粘贴代码。
For IOS, Select All and Copy code area, Add Bookmark this page. Click address bar, long press bookmark to edit, empty url and Paste.

步骤 3 / Step 3
在熊猫图片页运行书签解锁登录。
Click to run the bookmark to break-in panda page.


步骤 4 / Step 4
在专辑预览页运行书签启动列表模式。
Click to run the bookmark in a gallery preview page.


步骤 5 / Step 5
输入图片范围（起始,结束），填写“0”加载全部。（图片过多时加载缓慢）
Input picture range (from,to), leave "0" to load all. (Slow down under mass pictures)


步骤 6 / Step 6
是否加载原图？（需要下载权限且配额充足，更慢！）
Track orign? (Need download authority and enough quota, slower!)


步骤 7 / Step 7
开撸！（如需下载图片，页面加载完成后，保存网页，全部）
Enjoy it! (To download pictures, just save the Web Page, completely, after page load finished)
